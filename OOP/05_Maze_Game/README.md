# 🧩 Maze Game Console Application

## 📝 Description
A console-based Maze Game built using C#.
This project was developed as part of my Object-Oriented Programming (OOP) learning stage, focusing on game logic, classes, and continuous user interaction.

The game allows the player to move inside a maze using keyboard arrows while the maze is continuously drawn in the console.

---

## 🎮 Features
- Interactive console-based maze
- Player movement using keyboard arrow keys
- Continuous game loop
- Dynamic maze rendering

---

## 🛠 Technologies Used
- C#
- .NET Console Application

---

## 🧠 OOP Concepts Applied
- Classes and Objects
- Encapsulation
- Separation of responsibilities
- Game loop logic
- Object collaboration

---

## ▶️ How to Run
1. Open the solution file using Visual Studio
2. Run the application
3. Use the keyboard arrow keys to move the player inside the maze

Through this project, I practiced:
- Designing classes for game components

- Handling real-time user input in console applications
- Building continuous loops for interactive programs
