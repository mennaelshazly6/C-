# 🛒 Shopping System Console Application

## 📝 Description
A console-based Shopping System built using C# as part of the **C# Advanced** stage in my .NET learning journey.

The system supports two user roles:
- Employee: manages products (add, remove, update)
- Customer: browses products and manages a shopping cart

This project focuses on applying advanced C# and Object-Oriented Programming concepts in a real-life scenario.

---

## 🚀 Features

### 👨‍💼 Employee
- View product list
- Add new products
- Remove existing products
- Update product name or price

### 🧑 Customer
- View available products
- Add products to cart
- Remove products from cart
- View cart contents

---

## ▶️ How to Run
1. Open the solution file using Visual Studio
2. Run the application
3. Choose user type:
   - `E` for Employee
   - `C` for Customer
4. Follow the console menu options
