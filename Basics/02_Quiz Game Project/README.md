# 🧠 Quiz Game Console Application

## 📝 Description
A simple Quiz Game console application built using C#.
This project was developed as part of my C# learning journey to practice arrays, loops, methods, and exception handling.

The application asks the user a set of questions, takes input from the console, and checks whether the answers are correct.

---

## 🚀 Features
- Display multiple quiz questions
- Accept user answers through the console
- Validate user input
- Display correct or wrong answer feedback
- Handle empty input using exception handling

---

## 🛠 Technologies Used
- C#
- .NET Console Application

---

## 🧠 Concepts Applied
- Arrays
- Loops (`for`)
- Methods
- Console input/output
- Exception Handling (`try-catch`)
- Input validation

---

## ▶️ How to Run
1. Open the project solution using Visual Studio
2. Run the application
3. Answer the questions displayed in the console

---

## 🎯 Learning Outcome
Through this project, I practiced:
- Working with arrays to store questions and answers
- Using methods for logic separation
- Handling runtime errors using exceptions
- Validating user input in console applications
