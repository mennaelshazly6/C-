\# 📦 Inventory Management System



\## 🔹 Description

A simple console-based Inventory Management System built using C#.

This project is part of my C# learning journey and focuses on practicing basic programming concepts.



\## 🔹 Features

\- Add new products

\- Update existing product details

\- View all products

\- Menu-driven console interface



\## 🔹 Technologies Used

\- C#

\- .NET Console Application



\## 🔹 Concepts Applied

\- Arrays (2D Arrays)

\- Loops (while, for)

\- Switch-case statements

\- Methods

\- User input handling



\## 🔹 How to Run

1\. Open the solution file using Visual Studio

2\. Run the project

3\. Follow the menu options in the console



\## 🔹 Learning Outcome

This project helped me better understand:

\- Organizing logic into methods

\- Working with arrays

\- Building simple CRUD-like functionality
