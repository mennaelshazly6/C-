\# ✅ Task Tracker Console Application



\## 📝 Description

A simple Task Tracker console application built using C#.

This project was developed as part of my C# learning journey to practice basic programming concepts and console-based user interaction.



The application allows users to manage a list of tasks by adding, viewing, marking tasks as completed, and removing tasks.



\---



\## 🚀 Features

\- Add new tasks

\- View all tasks

\- Mark tasks as completed

\- Remove tasks from the list

\- Menu-driven console interface



\---



\## 🛠 Technologies Used

\- C#

\- .NET Console Application



\---



\## 🧠 Concepts Applied

\- Arrays

\- Loops (`while`, `for`)

\- Switch-case statements

\- Methods

\- Console input/output

\- Basic state management using variables



\---



\## ▶️ How to Run

1\. Open the project solution using Visual Studio

2\. Run the application

3\. Choose options from the menu (1–5) to manage tasks
